# Skypex's dotfiles

This is the dotfiles repository of Skypex.

## Desktop Setup

This repository contains dotfiles for the following programs:

- i3-wm
- rofi
- URxvt

## Installation

All dotfiles are deployed via symlinks.
The script *setup.sh* automates the symlink creation for a given list of files/folders.

So first, check out the *setup.sh* to verify everything is set up as you like it.

Afterwards just execute the script:

```bash
./setup.sh
```

Done!
